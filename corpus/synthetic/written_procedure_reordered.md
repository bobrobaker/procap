# LabRig start-up (reordered + gappy reference SOP)

A deliberately flawed 7-step written SOP used to demonstrate the OFFLINE content audit. It
documents the valve **before** the pump (reordered), **omits** the flow-rate step (gap), and
adds a **calibrate** step the video never shows (extra). An offline lexical audit of a
procedure with filled intents flags all three: out_of_order, missing_step, extra_in_doc.

1. Confirm the system is idle before start.
2. Open the main valve until valve open shows.
3. Start the feed pump and confirm pump running.
4. Calibrate the sensor array before first use.
5. Enable the heater and confirm heater on.
6. Set the temperature setpoint to 65 degrees.
7. Engage data logging to record the run.
