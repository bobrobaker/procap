# LabRig start-up procedure (written reference, deliberately gappy)

This is a sparse human-written procedure used to test the audit. It intentionally OMITS
the flow-setpoint step, so the audit should flag one missing step against a 4-step capture.

1. Confirm the system is idle (all indicators off).
2. Start the pump and confirm "PUMP RUNNING".
3. Open the valve and confirm "VALVE OPEN".
